#安装方法

　目前仅支持通达信软件使用，文件包中包含DLL、tne以及本说明文件。

　安装时，首先将CZSC.dll复制到通达信安装目录下的T0002\dlls目录之中，
并在通达信公式管理器中将本dll加载到1号dll插件之中，而后将CZSC.tne文
件导入到公式管理器之中即可使用，指标名称为CZSC。
